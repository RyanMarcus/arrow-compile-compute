Input matrix: compile vs execute 8

The JIT pays compile once per kernel shape (LLVM IR-gen + JIT), then every call is the warm execute cost. total = compile + execute is the cost of a single cold call; amortized over many calls it tends to execute. The verdict compares execute vs arrow. For dict inputs the JIT reads the encoding directly while arrow must decode first.
